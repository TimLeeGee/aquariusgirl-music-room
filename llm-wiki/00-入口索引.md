# Aquariusgirl Music Room LLM Wiki

#project #moc

這是 Aquariusgirl Music Room 的可追溯 Markdown 專案知識庫。它整理 2026-07-10 起工作樹中的文件與架構證據，不取代原始文件，也不宣稱涵蓋未讀取的歷史。當前版本為 0.1.51，source 與文件已同步 GitHub `main`，installer 仍只留本機；逐版歷史只讀 [CHANGELOG.md](../CHANGELOG.md)，artifact 證據由 [[12-來源文件索引]] 路由。

## 閱讀順序

1. 先讀 [[01-專案總覽]]、[[02-目前功能清單]]，理解產品與目前能力。
2. 任何改動前必讀 [[05-禁止事項]]、[[06-驗收標準]]、[[07-Agent工作規則]]。
3. 依任務追讀規格、踩坑、決策或發布頁；需要原始證據時用 [[12-來源文件索引]] 回查。
4. 逐版版本歷史只讀根目錄 [CHANGELOG.md](../CHANGELOG.md)；Wiki 只保留來源、決策與流程路標，不重複轉錄版本段落。

## 修改前必讀

- [ ] 確認任務不違反 [[05-禁止事項]]。
- [ ] 找出相關功能、規格、踩坑與驗收頁。
- [ ] 回查 [[12-來源文件索引]] 的原始證據；歷史紀錄不應覆蓋目前 source。
- [ ] 若影響程式行為，遵守 [[07-Agent工作規則]] 的核准閘門。

## 修改後必更新

- [ ] 功能或使用者期待變了，更新 [[02-目前功能清單]] 與 [[03-功能規格與使用者期待]]。
- [ ] 新回歸、限制或驗證缺口，更新 [[04-已知問題與踩坑紀錄]] 與 [[06-驗收標準]]。
- [ ] 新決策或取捨，更新 [[09-歷史決策紀錄]]。
- [ ] 文件來源或狀態改變，更新 [[12-來源文件索引]] 與 [[99-更新日誌]]。
