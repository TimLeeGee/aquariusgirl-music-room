## 產品定位

Aquariusgirl Music Room 是 React、TypeScript、Vite 與 Electron 製作的本地優先音樂播放器。它只處理使用者明確選擇的本機音樂，Web 版可在瀏覽器使用，Electron 版可產出 macOS DMG 與 Windows NSIS 安裝檔。

## 技術架構

| 層次 | 主要位置 | 責任 |
| --- | --- | --- |
| Renderer | src/ | React 播放器、列表、設定、AI 面板與互動 |
| 播放控制 | src/hooks/useAudioPlayer.ts | audio 元素、播放狀態、檔案寫回暫停與恢復 |
| 資料保存 | src/storage/indexedDb.ts、src/hooks/usePlaylists.ts | tracks metadata、播放清單與使用者設定；不保存音樂本體 |
| Electron | electron/main.ts、electron/preload.ts | 最小 IPC、視窗、原生檔案對話框、桌面能力 |
| Metadata | electron/songInfoWriter.ts、src/components/SongInfoPanel.tsx | 原始檔文字與封面寫回、讀回驗證 |
| AI | src/ai/、src/hooks/useLocalAI.ts、electron/ai/ | 僅使用已載入歌曲的安全 metadata 與離線 runtime |
| 發行 | package.json、scripts/、release-delivery/ | check、DMG/EXE、checksum 與 QA 交付 |

## 核心方向

- 本地優先：不串流、不下載、不掃描整台硬碟。
- 資料最小化：不將 File、Blob、ArrayBuffer 或 object URL 寫入 localStorage 或 IndexedDB。
- 大曲庫可用：維持既有 visible-window 清單與窄範圍更新。
- 寫回可驗證：原始檔成功、讀回、全域 track 更新與單曲 metadata 保存都完成後才可回報成功。

## 目前重點
